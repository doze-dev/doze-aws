# S3 — API support

Tiers: **F** = functional · **C** = cosmetic round-trip · **S** = honest stub.

Both addressing styles (path + virtual-hosted), both SDK generations, and the
full upload-body matrix: plain, UNSIGNED-PAYLOAD, signed aws-chunked
(STREAMING-AWS4-HMAC-SHA256-PAYLOAD), and trailer-checksum streaming
(STREAMING-*-TRAILER — aws-sdk-go-v2's default). Object bodies are streamed
end to end; memory use is independent of object size.

| Operation | Tier | Notes |
|---|---|---|
| CreateBucket / DeleteBucket / HeadBucket / ListBuckets / GetBucketLocation | F | naming rules enforced; delete requires empty |
| PutObject / GetObject / HeadObject | F | metadata, standard headers, storage class; conditional reads (If-Match/None-Match/(Un)Modified-Since) AND conditional writes (If-None-Match:*, If-Match); single-range requests (multi-range → 200 full body, documented) |
| Flexible checksums | F | CRC32, CRC32C, CRC64NVME, SHA1, SHA256; header- or trailer-declared, verified server-side; echoed with x-amz-checksum-mode; SDK client-side validation passes |
| Content-MD5 | F | verified |
| DeleteObject / DeleteObjects | F | delete markers on versioned buckets, governance bypass header |
| CopyObject | F | metadata/tagging directives, copy-source conditionals, versioned sources |
| ListObjects (V1) / ListObjectsV2 | F | prefix, delimiter/CommonPrefixes, paging (marker / continuation token), fetch-owner |
| Bucket versioning | F | Enabled/Suspended, null versions, ListObjectVersions with markers, get/delete by versionId |
| Multipart | F | Create/UploadPart/UploadPartCopy (with source ranges)/Complete/Abort/ListParts/ListMultipartUploads; part-size + order validation; "-N" composite ETags; COMPOSITE and FULL_OBJECT checksum types |
| Object tagging | F | put/get/delete, x-amz-tagging header, tagging-count |
| GetObjectAttributes | F | ETag, Checksum, ObjectSize, StorageClass |
| CORS | F | config CRUD + real preflight evaluation (origins with wildcards, methods, headers, expose, max-age) and response decoration |
| Lifecycle | F | Expiration (Days), NoncurrentVersionExpiration, AbortIncompleteMultipartUpload — enforced by janitor; storage-class transitions accepted, never acted on (one storage class locally) |
| Object lock | F | bucket config, retention (GOVERNANCE with bypass / COMPLIANCE), legal hold — deletion genuinely blocked |
| Bucket website | F | config CRUD + index/error document serving on directory requests and 404s |
| Presigned URLs | F | SigV2 + SigV4 forms accepted, expiry enforced (signature not verified — local emulator) |

| Encryption / Logging / Notification / Replication / Accelerate / RequestPayment configs | C | stored and returned (SSE headers echoed on objects); notification wiring becomes functional in Phase 6 |
